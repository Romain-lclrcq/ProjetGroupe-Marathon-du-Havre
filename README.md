# ProjetGroupe-Marathon-du-Havre

Créer un formulaire pour permettre les inscriptions au marathon du Havre

inscription form
Vous veillerez a corectement structurer votre code, vous respecterez les normes de developpement (utilisation de l'anglais pour les nom de variable, fonction etc) n'hésitez pas à commenter vos codes

ex 1 :
pour le tout nouveau marathon de la ville du havre vous allez devoir réaliser un site internet pour s'inscrire à l'évènement.
En équipe de trois vous réaliserez un formulaire d'inscription qui demande
le nom _
le prénom _
l'age _
mail _
téléphone _
le formulaire demandera ensuite si le participant souhaite réaliser un semi-marathon ou un marathon classique _
en fonction du type de course choisi vous afficherait sur la même page un panier qui sera de 90€ pour le semi et 130€ pour le complet

vous vérifirez que l'utilisateur a bien remplie tous les champs et au bon format (ex: numéro de téléphone en +33XXXXXXXXX, XXXXXXXXXX; mail sous la forme xxx@xx.xx)

ex 2 :
Vous ajouterez la possibilité de s'inscrire en équipe, pour cela le formulaire devra comprendre AU MINUMUM deux inscription à remplir.
Ces deux inscriptions auront les même champs que l'inscription précédente, le premier participant sera concidéré comme le capitaine de l'équipe.
Vous actualiserez le panier en détaillant le cout de chaque inscription et afficher le prix total

ex 3 :
En dessus de la dernière inscription vous ajouterez un bouton pour ajouter un nouveau participant à une équipe. en cliquant sur ce bouton tout les champ liée a une incsription seront ajouter comme pour le 2 eme utilisateur.
Vous actualiserez le panier en prenant en compte chacun des participations.

ex4 :
Vous ajouterez la posibilité de supprimer un participant
vous ferez en sorte que le bouton pour ajouter un participant sera désativé si l'utilisateur n'a pas rempli correctement toutes les inscriptions précedentes.
